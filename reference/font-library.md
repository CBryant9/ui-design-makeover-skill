# Font library

Curated catalogue of fonts known to work for app/dashboard design in 2026, with personality markers, licensing, and dashboard suitability notes. Used by Phase 3 to generate the 10-pairing proposals.

## How to use this library

For a chosen direction, pick **3–4 display options + 3–4 body options + 1–2 mono options** from the library that fit the direction, then combine into 10 pairings. Never propose the same pairing twice; vary display type across the 10.

## Display / heading fonts — with character

### Serifs (editorial / magazine / warm feel)

- **[Tobias](https://displaay.net/typeface/tobias)** — Baroque/transitional serif with diagonal stress and abundant alternates. Magazine gravity, contemporary voice. **Premium** ($260+). Best for: editorial, warm magazine, quiet luxe.
- **[Migra](https://pangrampangram.com/products/migra)** — Sharp, spiky serif with fine terminals. Editorial, fashion-leaning, deliberate energy. **Free to try** (paid for commercial). Best for: editorial with personality, warm magazine, art-forward.
- **[PP Editorial New](https://pangrampangram.com/products/editorial-new)** — Modern editorial serif, very current in design-forward apps. **Free to try** (paid for commercial). Best for: editorial, quiet luxe, modern publication. Risk: trendy right now, may feel dated in 2-3 years.
- **[Fraunces](https://fonts.google.com/specimen/Fraunces)** — Soft, characterful variable serif with playable axes (SOFT, WONK). **Free (Google Fonts).** Best for: warm organic, personal/friendly, editorial-with-heart.
- **[Newsreader](https://fonts.google.com/specimen/Newsreader)** — Readable editorial serif optimised for screens, supports multiple optical sizes. **Free (Google Fonts).** Best for: editorial, reading-heavy apps, warm magazine.
- **[Heldane](https://klim.co.nz/retail-fonts/heldane/)** — Razor-sharp transitional serif by Klim. Pairs beautifully with quiet sans-serifs. **Premium** ($200+). Best for: premium editorial, quiet luxe, publication-forward.
- **[Playfair Display](https://fonts.google.com/specimen/Playfair+Display)** — Classic high-contrast serif. **Free (Google Fonts).** Best for: traditional editorial. **Risk:** overused, feels "stock editorial" — use only if no alternatives fit.
- **[Source Serif 4](https://fonts.google.com/specimen/Source+Serif+4)** — Adobe-designed editorial serif, very readable. **Free (Google Fonts).** Best for: editorial, documentation, reading-heavy.

### Sans-serifs (characterful display)

- **[Cabinet Grotesk](https://www.fontshare.com/fonts/cabinet-grotesk)** — Characterful neo-grotesque with personality details. **Free (Fontshare).** Best for: playful distinct, editorial-with-personality, modern minimal with flavour.
- **[PP Neue Montreal](https://pangrampangram.com/products/neue-montreal)** — Minimal, refined sans-serif. Functional but distinctive. **Free to try** (paid for commercial). Best for: modern minimal, quiet luxe, Apple-clean.
- **[Mabry](https://www.colophon-foundry.org/custom-fonts/mabry)** — Geometric sans with charming proportional moments that make brands feel human. **Premium.** Best for: modern minimal with warmth, product-brand feel.
- **[Founders Grotesk](https://klim.co.nz/retail-fonts/founders-grotesk/)** — Neutral but characterful workhorse. **Premium** ($200+). Best for: quiet brutalism, editorial, tech-precise.
- **[Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk)** — Proportional sister of Space Mono, retains idiosyncratic details. **Free (Google Fonts).** Best for: tech-precise, slightly retro-futuristic. **Risk:** very heavily used 2022–2024, may feel dated.
- **[GT Walsheim](https://www.grillitype.com/typeface/gt-walsheim)** — Friendly, contemporary geometric sans. **Premium.** Best for: warm modern, product brand, approachable.
- **[Söhne](https://klim.co.nz/retail-fonts/soehne/)** — Workhorse grotesk with subtle warmth. Used by OpenAI, Mercury. **Premium.** Best for: modern minimal, tech-confident, product-forward.
- **[Monument Grotesk](https://abcdinamo.com/typefaces/monument-grotesk)** — Compressed statement typography. **Premium.** Best for: editorial statements, bold moments, poster-feel.

### Monospace display (deliberate "computer-y" feel)

- **[Departure Mono](https://departuremono.com/)** — Pixel-y, retro-futuristic. **Free.** Best for: distinctive/playful, retro-futuristic, deliberately weird.
- **[Berkeley Mono](https://berkeleygraphics.com/typefaces/berkeley-mono/)** — Premium, distinctive, technical-feeling. **Premium.** Best for: tech console, power-user tools, technical precision.

## Body / UI sans-serifs (clean, legible)

- **[Switzer](https://www.fontshare.com/fonts/switzer)** — Neo-grotesk with high x-height (79%), excellent for data-heavy dashboards. **Free (Fontshare).** Dashboard suitability: **excellent** — high x-height, good numerals.
- **[General Sans](https://www.fontshare.com/fonts/general-sans)** — Inter alternative with more character. **Free (Fontshare).** Dashboard suitability: **good**.
- **[Manrope](https://fonts.google.com/specimen/Manrope)** — Semi-geometric, beautifully kerned numbers, great for fintech/dashboards. **Free (Google Fonts).** Dashboard suitability: **excellent** — numerals are its standout feature.
- **[Geist Sans](https://vercel.com/font)** — Vercel's Swiss-influenced sans. **Free.** Dashboard suitability: **good**. **Risk:** every Vercel-stack project uses it; if your brief is "escape the look of AI-generated / deployed-on-Vercel apps," pick something else.
- **[DM Sans](https://fonts.google.com/specimen/DM+Sans)** — Clean, low-personality, perfect supporting role. **Free (Google Fonts).** Dashboard suitability: **good**.
- **[Mona Sans](https://github.com/github/mona-sans)** — GitHub's variable workhorse. **Free.** Dashboard suitability: **good**.
- **[Satoshi](https://www.fontshare.com/fonts/satoshi)** — Geometric, slightly warm. **Free (Fontshare).** Dashboard suitability: **good**.
- **[Söhne Mono](https://klim.co.nz/retail-fonts/soehne-mono/)** — Used in body mode, gives technical/editorial hybrid. **Premium.** Dashboard suitability: **niche** (mono bodies reduce reading speed).

- **[IBM Plex Mono](https://fonts.google.com/specimen/IBM+Plex+Mono)** — Refined corporate-but-warm mono. **Free (Google Fonts).** Best for: IBM-adjacent corporate warmth, technical labels.

## Dashboard-specific notes

For data-heavy apps (dashboards, analytics, any interface with tables or KPI cards):

- **Numerals matter.** The body font's numerals should be either:
  - Tabular (same width) for tables — easier columns align
  - Beautifully proportional (not the same width) for metric cards — better visual rhythm
  - Both, if the font supports it via OpenType features
- **High x-height helps at small sizes.** Switzer and Manrope are strong here.
- **Avoid narrow/condensed body fonts.** UI legibility drops.
- **Display fonts at 14px are usually unreadable.** Set minimum size for display typefaces (e.g. only use Tobias at 24px+).

## Don't pair with

For any distinctive display choice, the body should be invisible/workhorse — DM Sans, Switzer, Manrope, General Sans. Don't pair two personality fonts.

For any characterful body choice (Fraunces as body, Söhne Mono as body), the display must be even quieter. Most body fonts are already the "quiet" side of the pairing.

## Bans (repeat for emphasis)

**Never propose as primary:**
- Inter
- Roboto
- Arial / Helvetica
- system-ui
- San Francisco / SF Pro (Apple system font)
- Segoe UI

Use these only as a **body fallback** in the CSS stack if the chosen webfont fails to load. Never as the design choice.
